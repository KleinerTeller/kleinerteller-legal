2. Welche Daten werden verarbeitet?

Im Rahmen der Nutzung der App „Kleiner Teller" werden folgende personenbezogene Daten erhoben und verarbeitet:

  • E-Mail-Adresse (für Registrierung und Login)
  • Kinderdaten: Name, Geburtsdatum, Alter, Allergien
  • Ernährungsdaten: Mahlzeiten-Protokolle, eigene Rezepte
  • Wachstumsdaten: Gewicht, Größe, Kopfumfang
  • Geräteinformationen (für Benachrichtigungen)

3. Zweck der Datenverarbeitung

Die Daten werden ausschließlich zur Bereitstellung der App-Funktionen verwendet:

  • Speicherung und Synchronisation deiner Kindsdaten
  • Anzeige personalisierter Ernährungsempfehlungen
  • Verwaltung des Premium-Abonnements

4. Datenspeicherung

Die Daten werden auf Servern von Google Firebase (Google LLC, USA) gespeichert. Die Übertragung erfolgt verschlüsselt. Google LLC ist unter dem EU-US Data Privacy Framework zertifiziert.

Für die Verwaltung von In-App-Käufen wird RevenueCat, Inc. (USA) eingesetzt. RevenueCat verarbeitet ausschließlich anonymisierte Kaufdaten.

5. Datenweitergabe

Deine Daten werden nicht an Dritte verkauft oder zu Werbezwecken weitergegeben.

6. Datenlöschung

Du kannst dein Konto und alle gespeicherten Daten jederzeit durch Kontaktaufnahme unter info@kleinerteller.de löschen lassen.
